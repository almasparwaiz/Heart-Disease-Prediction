## Heart Disease Prediction – ML Web App

A practical and interactive Machine Learning web application that predicts the risk of heart disease based on user-provided medical and lifestyle information. Built with a focus on simplicity, accuracy, and real-world usability.

### Problem Statement

Heart disease is one of the leading causes of death worldwide, and early detection plays a critical role in prevention. However, many people are unaware of their risk level until it becomes severe.

This project aims to bridge that gap by using machine learning to provide an instant and accessible risk prediction tool based on basic health inputs.

### My Approach (What I Built)

I developed an end-to-end machine learning solution that:

Collects patient health data through a simple UI
Processes and prepares data for prediction
Uses a trained ML model to estimate heart disease risk
Returns instant results as High Risk / Low Risk

The goal was to make the system lightweight, fast, and easy for anyone to use without technical knowledge.

### Features

Clean and user-friendly Streamlit interface
Real-time heart disease risk prediction
Simple input form for medical attributes
Instant classification result (Low Risk / High Risk)
Fast and responsive performance
